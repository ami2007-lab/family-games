# 🎮 דייזי·גיימס — הארקייד האישי

אתר משחקי מסיבה לבת המצווה של דייזי — שבעה משחקים, עובד גם בלי אינטרנט, ניתן להתקנה כאפליקציה.

🔗 **לשחק:** https://wiezmannthe1st.github.io/Ariel-Games/

## המשחקים
🗣️ **אליזי** · ❓ **21 שאלות** · 🌍 **ארץ עיר** · 🧠 **אמת או שקר** · ⚖️ **זה או זה** · 🎵 **מצא שיר** · 🤫 **סינכרון** (כולל מצב מרובה טלפונים)

## תכונות
- 📱 התקנה כאפליקציה (PWA) + עבודה אופליין
- 🏆 שיאים והיסטוריית משחקים
- ☁️ בנק מילים משותף ומשחק מרובה מכשירים (דורש הגדרת Firebase)
- 📶 קוד QR מובנה לשיתוף מהיר
- 🎯 רמות קושי
